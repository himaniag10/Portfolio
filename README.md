# 🚀 Himani Portfolio

A futuristic space-themed developer portfolio built using React, Vite, TailwindCSS, and Framer Motion.

## 🌌 Features

* Dark futuristic UI
* Animated hero section
* Smooth scrolling navigation
* Responsive design
* Glassmorphism cards
* Skills showcase
* Certifications section
* Achievements section
* Resume download option
* GitHub & LinkedIn integration
* Modern portfolio layout

# 📂 Project Setup

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/himani-portfolio.git
```

## Open Project

```bash
cd himani-portfolio
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```
